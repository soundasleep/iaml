The \modelProperty{Map}{field value} of the \modelLink{Map} is used as the location for the rendered map. If the \modelProperty{Map}{field value} changes, the location of the map will be refreshed to the new location. If the \modelProperty{Map}{field value} of the \modelLink{Map} is empty, the map will revert to a default location. When the current location of the rendered map changes, the \modelProperty{Map}{field value} of the \modelLink{Map} should also change, and the \event{onChange} \modelLink{Event Trigger} will be triggered\issue{225}.
