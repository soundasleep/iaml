The \modelProperty{Map Point}{field value} of the \modelLink{Map Point} is used as the location of the current point. If the \modelProperty{Map Point}{field value} changes, the rendered map point will be refreshed to the new point. If the \modelProperty{Map Point}{field value} of the \modelLink{Map Point} is empty, the map point will not be displayed. If the \modelLink{Map Point} is moved to a different location, the \modelProperty{Map Point}{field value} of the \modelLink{Map Point} should also change, and the \event{onChange} \modelLink{Event Trigger} will be triggered\issue{225}.
